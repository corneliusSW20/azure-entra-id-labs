# Azure Administrator Labs Portfolio

This repository contains hands-on Microsoft Azure labs completed
as part of Azure Administrator (AZ-104) learning.

## Labs Included

| Lab | Topic | Key Skills |
|----|------|-----------|
| Lab 01 | Microsoft Entra ID Identities | Users, Groups, Dynamic Membership |
| Lab 02 | RBAC and Subscriptions | Roles, Access Control |
| Lab 03 | Azure Networking | VNets, Subnets, NSGs |
| Lab 04 | Virtual Machines | VM Deployment, Management |

## Technologies Used
- Microsoft Azure
- Microsoft Entra ID
- Azure Portal
- Azure CLI

## Goal
Build real-world Azure administration skills through practical labs.
