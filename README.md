# Azure Administrator Labs Portfolio (AZ-104)

This repository contains a collection of hands-on Microsoft Azure labs
completed as part of Azure Administrator (AZ-104) training.
The labs focus on real-world Azure administration tasks including identity,
governance, networking, compute, storage, and monitoring.

Each lab is documented with objectives, implementation steps, and outcomes
to demonstrate practical Azure experience.

---

## Lab Index

### Administer Identity
- **Lab 01: Manage Microsoft Entra ID Identities**  
  Provision users and groups in Microsoft Entra ID and configure dynamic
  group membership to automate access management.

### Administer Governance and Compliance
- **Lab 02a: Manage Subscriptions and RBAC**  
  Manage Azure subscriptions, assign built-in roles, and implement
  Role-Based Access Control (RBAC).

- **Lab 02b: Manage Governance via Azure Policy**  
  Create and assign Azure Policies to enforce organizational standards
  and ensure compliance.

### Administer Azure Resources
- **Lab 03: Manage Azure Resources by Using ARM Templates**  
  Deploy and manage Azure resources using Azure Resource Manager (ARM)
  templates for consistent and repeatable infrastructure.

### Implement Virtual Networking
- **Lab 04: Implement Virtual Networking**  
  Configure virtual networks, subnets, network security groups, and
  DNS settings.

### Administer Intersite Connectivity
- **Lab 05: Implement Intersite Connectivity**  
  Implement VNet peering and configure site-to-site connectivity
  between Azure networks.

### Administer Network Traffic Management
- **Lab 06: Implement Network Traffic Management**  
  Manage inbound and outbound traffic using load balancers and
  traffic management solutions.

### Administer Azure Storage
- **Lab 07: Manage Azure Storage**  
  Configure storage accounts, blob containers, file shares,
  and access controls.

### Administer Virtual Machines
- **Lab 08: Manage Virtual Machines**  
  Deploy, configure, monitor, and manage Azure virtual machines,
  including availability and scaling options.

### Administer PaaS Compute Options
- **Lab 09a: Implement Web Apps**  
  Deploy and manage Azure App Services and web applications.

- **Lab 09b: Implement Azure Container Instances**  
  Run containerized workloads using Azure Container Instances.

- **Lab 09c: Implement Azure Container Apps**  
  Deploy and manage container-based applications using Azure Container Apps.

### Administer Data Protection
- **Lab 10: Implement Data Protection**  
  Configure backups and recovery using Azure Backup and Recovery Services Vaults.

### Administer Monitoring
- **Lab 11: Implement Monitoring**  
  Monitor Azure resources using Azure Monitor, Log Analytics,
  and alert rules.
